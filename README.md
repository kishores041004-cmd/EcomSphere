# 🛒 EcomSphere - Full Stack E-Commerce Web Application

EcomSphere is a comprehensive full-stack e-commerce platform developed to provide a seamless online shopping experience. The application allows customers to browse products, manage shopping carts, place orders, and manage their profiles, while administrators can efficiently handle products, inventory, and customer orders through a dedicated admin dashboard.

---

## 🚀 Features

### 👤 Customer Features
- User Registration & Secure Login
- Product Browsing & Searching
- Category-wise Product Listing
- Product Details View
- Add to Cart & Cart Management
- Order Placement
- User Profile Management
- Responsive User Interface

### 🔐 Admin Features
- Admin Authentication
- Product Management (Add, Update, Delete)
- Category Management
- Inventory Monitoring
- Order Management

- Dashboard Overview

## ⚙️ Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/kishores041004-cmd/EcomSphere.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd EcomSphere
```

### 3️⃣ Setup MySQL Database

Create a database:

```sql
CREATE DATABASE ecomsphere;
```

Import the SQL file into MySQL Workbench.

### 4️⃣ Configure Database Connection

Update database credentials in your database configuration file:

```java
String url = "jdbc:mysql://localhost:3306/ecomsphere";
String username = "root";
String password = "your_password";
```

### 5️⃣ Run the Project

- Import project into Eclipse/IntelliJ IDEA
- Configure Apache Tomcat Server
- Deploy the project
- Start the server

Open in browser:

```text
http://localhost:5173/
```
